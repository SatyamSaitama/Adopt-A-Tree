

## Installation

In each service directory, install the required packages using:

```bash
npm install
```

### Individual Service Entry Points

- **ProduceService:**
  ```bash
  cd ProduceService
  npm install
  node index.js
  ```

- **TreeService:**
  ```bash
  cd TreeService
  npm install
  node index.js
  ```

- **UserService:**
  ```bash
  cd UserService
  npm install
  node index.js
  ```

## Running the API Gateway

To run the API Gateway, execute:

```bash
npm install
node main.js
```
